Generate Packing Slips

1. Generate Box Numbers

We will now generate packing slips, each of which will be assigned a package number. Why do we need package numbers?

When batch picking, it's necessary to distribute the items into separate boxes afterward. It's challenging to differentiate boxes using order numbers or tracking numbers.

Instead, short and clear numbers like 1, 2, etc., are more convenient for this task.

Click the button in the sidebar to create box numbers in the   Orders   sheet.

2. Generate Packing Slips

Simultaneously with the generation of package numbers, all packing slips for each box will be created in the   Packing Slips  .

Since the pages are already divided, you can print them directly.

3. Packing Slip Template

You can modify the packing slip format in the   Packing Slip Template  .

For example, let's make the following changes:

Change the box tag from "HappyChannel" to "Online"

and add your logo image. (Make sure to "Place logo image in Cell" and not "Place over cells.")

Here is what the modified template looks like.

After making these changes, generate the packing slips again, and you'll see them displayed in the   Packing Slips   as shown on the right.
